# Sanctuary
## A Fork Of [Haven](https://github.com/ancsemi/Haven)

Currently, this repository is used to fit the needs of, "Sanctuary"- A Haven instance I, ArtyDaSmarty, hosts. It is not aimed to be maintained for others, but if any like the differences, they can use it!

## Current Differences To The Main Haven:

- Proxy/Persona System Built-In With Some QOL
- Forum Channels
- S3 Volume Option For Uploads/Attachments (Use The S3 Branch)
